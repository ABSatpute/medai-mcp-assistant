# Unused Files Archive

This folder contains files that are not used by the main Flask application (`app.py`) but are kept for reference.

## Alternative Implementations
- `mcp_app.py` - Streamlit version of the medical assistant
- `flask_app.py` - Earlier Flask implementation (superseded by app.py)
- `simple_app.py` - Basic Flask version
- `web_app.py` - Another alternative implementation

## Test & Debug Files
- `test_server.py` - Simple test server for debugging
- `test_db_connection.py` - Database connection testing utility
- `test_connection.html` - HTML test file
- `test_sidebar.html` - UI component test file

## Unused Components
- `prescription_server.py` - MCP server (moved to direct function in app.py for performance)
- `prescription_server.py.backup` - Backup of prescription server
- `fixed_prompts.py` - Unused prompt definitions
- `requirements.txt` - General requirements (use requirements_flask.txt instead)

## Utility Scripts
- `run_app.py` - App launcher script

## Note
These files can be restored if needed but are not required for the main application functionality.
